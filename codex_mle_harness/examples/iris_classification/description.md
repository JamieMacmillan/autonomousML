# Iris Classification Mini Benchmark

Build a small supervised classifier for the Iris flower dataset.

The read-only `input/` directory contains:

- `train.csv`: labeled rows with sepal and petal measurements.
- `test.csv`: unlabeled rows with the same feature columns and an `id`.

Write an executable `run.py` that creates `submission/predictions.csv` with columns:

```csv
id,species
```

Valid species labels are `setosa`, `versicolor`, and `virginica`.
